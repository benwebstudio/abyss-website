# Abyss Website

A cinematic deep-ocean React website built with Vite, TypeScript-friendly React components, Tailwind CSS, and premium scroll/hover interactions.

## Local setup

```bash
npm install
npm run dev
```

The local dev server runs through the web app workspace.

## Build

```bash
npm run build
```

## Notes

- Source lives in `apps/web`.
- Generated build output and dependencies are intentionally excluded from Git.
- Video and image assets are included because they are required for the cinematic Abyss experience.
